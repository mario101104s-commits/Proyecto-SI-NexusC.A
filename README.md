# Nexus C.A. - Sistema Integrado

🔗 **Acceso Público al Sistema:**
[https://mjsanchezr.github.io/Proyecto-SI-NexusC.A/](https://mjsanchezr.github.io/Proyecto-SI-NexusC.A/)

**Nexus C.A.** representa el desarrollo exclusivo del componente *front-end* de un Sistema de Planificación de Recursos Empresariales (ERP) como una intranet empresarial, completamente funcional y escalable. Este proyecto surge como el pilar fundamental para la modernización tecnológica de la infraestructura, implementando una suite integral de módulos departamentales interconectados (como recursos humanos, inventario, finanzas, operaciones). Su objetivo primordial es centralizar, optimizar y automatizar los flujos de trabajo críticos de la empresa, solucionando de manera efectiva problemas sistémicos de comunicación interdepartamental, cuellos de botella en la gestión de datos, y redundancia operativa. Al aplicar sistemas modernos de gestión de información como (ERP, CRM, SCM, BI, LMS, SFA) y conceptualizar arquitecturas centradas en el usuario, este ERP dota a los administradores de un panel de control intuitivo y analítico para la toma de decisiones en tiempo real, impulsando significativamente la competitividad y eficiencia operativa de la organización.

La aplicación web fue desarrollada utilizando **React** junto con **TypeScript** y **Vite**, brindando una base moderna, fuertemente tipada y con un rendimiento excepcional. La interfaz gráfica fue diseñada aplicando **Tailwind CSS**, lo que permitió construir un sistema responsive, con un diseño limpio y visualmente atractivo. Para asegurar una experiencia de usuario interactiva y plenamente accesible, se usaron los componentes modulares de **Radix UI**, combinados con animaciones fluidas o micro-interacciones impulsadas por **Framer Motion**. Además, la representación visual de datos complejos se maneja a través de **Recharts**. A nivel estructural, el código del proyecto fue organizado estrictamente bajo una arquitectura funcional basada en componentes, separando de manera limpia la presentación visual de la lógica de estado y gestión de formularios (manejada de forma robusta con **React Hook Form**), garantizando así un código limpio, altamente modular, fácil de mantener y preparado para escalar sin problemas.
